# Lung-Breathing-Simulator-A-cgip-project
A Computer Graphics mini project that simulates the human lung breathing mechanism using DDA line algorithm, Midpoint Circle algorithm, and polygon rendering in Python (Pygame).

Lung Breathing Mechanism Simulator
Project Description

This project simulates the human respiratory system using computer graphics techniques. It demonstrates inhalation and exhalation using animation.

Algorithms Used
DDA Line Drawing Algorithm (for bronchi)
Midpoint Circle Algorithm (for trachea rings)
Polygon Rendering (for lungs and diaphragm)
Technologies Used
Python
Pygame
Features
Realistic lung animation
Dynamic breathing simulation
Algorithm-based rendering
How to Run
Install pygame:
pip install pygame
Run:
python cgip.py
